Worked with peers for input and advice. I determined the best way to approach this project. For the fronted, I used HTML to create the grid layout and JavaScript to make the grid elements interactive. CSS was used for styling. For the backend portion, I used JavaScript to make an algorithm that iterates through each grid to find the possible moves and pick the best move to make. The move is made and the process repeats until we reach the destination.
